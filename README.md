## 20/08/15 JsonPractice

20/08/15 Json연습은 바로 DTO안에 또다른 DTO의 값을 넣는 것입니다.

예제는

```java
{
    "name" : "f",
    "id" : "flamme1004",
    "inner" : {
        "number" : "010-0000-0000",
        "address" : "강남구"
    }
}

```

으로 했으며 값은 잘 들어갔습니다.

자바스크립트에서 JSONStringfy를 할때 객체안에 객체를 넣어주고 말아주면 될 듯 합니다.

이상입니다.
